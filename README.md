# Zoo Management System

Невеликий консольний проєкт для управління зоопарком, побудований на модульній архітектурі. Система дозволяє керувати обліковими записами співробітників та вести базу даних тварин.

## Вимоги

* `g++` з підтримкою **C++11** або новішого стандарту
* Linux / macOS / Windows (будь-яке середовище з терміналом)

## Структура проєкту

```text
zoo-system/
├── main.cpp          // Основний код програми (entry point)
├── users.txt         // База даних користувачів
├── animals.txt       // База даних тварин
└── report.txt        // Згенерований звіт
How to compile
Для компіляції необхідно зібрати основний файл проєкту:
g++ main.cpp -o zoo_system
Запуск
Після успішної компіляції:
./zoo_system
Модулі
Auth — модуль авторизації та реєстрації (Login/Register)

AddAnimal — підсистема додавання нових тварин з валідацією ID

UpdateAnimal — редагування даних (вік, стан здоров'я)

Reporting — генерація статистичного звіту по видах тварин
Дані для входу (за замовчуванням)
При першому запуску система створює наступних користувачів:
Роль,Логін,Пароль
Administrator,admin,adminpass
Director,director,directorpass
Inspector,inspector1,inspectorpass
Veterinarian,vet1,vetpass
